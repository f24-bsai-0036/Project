This directory contains coursework and the semester project completed for the **Data Structures & Algorithms** course.

This course focused on efficient data organization, algorithm design, and problem-solving using **C++**.

Topics covered include:

- Arrays
- Linked Lists
- Stacks
- Queues
- Trees
- Graphs
- Searching Algorithms
- Sorting Algorithms
