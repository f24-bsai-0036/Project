# Project Vault

## Overview

**Project Vault** is a simple banking system developed in **8086 Assembly Language** as the semester project for the Computer Organization & Assembly Language course.

The project demonstrates low-level programming concepts, memory management, and user interaction using Assembly Language.

## Features

- Account management
- Deposit and withdrawal operations
- Menu-driven interface
- Assembly Language implementation

## Technologies Used

- 8086 Assembly Language
- emu8086

## Learning Outcomes

- Assembly Language Programming
- Computer Architecture
- Memory Organization
- Low-Level Programming
