# 8086 Assembly Programs

## Overview

This directory contains a collection of **8086 Assembly Language** programs developed during the **Computer Organization & Assembly Language** course.

These programs were created using **emu8086** to practice and understand low-level programming concepts, processor instructions, memory operations, arithmetic, loops, procedures, and interrupt handling.

## Topics Covered

- Input and Output
- Arithmetic Operations
- Conditional Statements
- Loops
- Arrays
- Strings
- Procedures
- Stack Operations
- Number Conversion
- Macros
- Interrupts
- Memory Addressing

## Development Environment

- 8086 Assembly Language
- emu8086

> These programs were developed for educational purposes as part of my undergraduate studies in Artificial Intelligence at Riphah International University.
