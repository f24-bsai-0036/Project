# Silent Voice System

## Overview

**Silent Voice System** is a hand gesture recognition system developed in **Python** that converts recognized hand signs into spoken audio.

The project demonstrates the application of computer vision and artificial intelligence to assist communication.

## Features

- Hand gesture recognition
- Speech output
- Real-time detection
- AI-powered recognition

## Technologies Used

- Python
- PyCharm

## Learning Outcomes

- Computer Vision
- Artificial Intelligence
- Image Processing
- Python Programming
