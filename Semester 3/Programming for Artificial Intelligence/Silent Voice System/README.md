# Silent Voice System

## Overview

**Silent Voice System** is a hand gesture recognition application developed in **Python** as the semester project for the **Programming for Artificial Intelligence** course.

The system recognizes hand gestures through computer vision techniques and converts the detected signs into spoken audio, demonstrating an AI-assisted communication solution.

## Features

- Hand gesture recognition
- Real-time gesture detection
- Speech output
- AI-powered sign interpretation
- User-friendly interface

## Technologies Used

- Python
- PyCharm
- OpenCV
- MediaPipe
- Text-to-Speech (TTS)
