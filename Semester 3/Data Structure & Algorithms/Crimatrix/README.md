# Crimatrix

## Overview

**Crimatrix** is a console-based Crime Record Management System developed in **C++** as the semester project for the **Data Structures & Algorithms** course.

The application provides an organized way to manage crime records while demonstrating the implementation of fundamental data structures and algorithms.

## Features

- Add crime records
- Search crime records
- Update existing records
- Delete records
- Display all records
- Menu-driven console interface

## Technologies Used

- C++
- Embarcadero Dev-C++

## Learning Outcomes

Through this project, I gained practical experience with:

- Data Structures
- Searching Algorithms
- Sorting Techniques
- Dynamic Data Management
- Algorithmic Problem Solving
- Modular Programming
