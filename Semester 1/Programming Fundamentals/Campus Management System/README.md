# Campus Management System

## Overview

The **Campus Management System** is a console-based application developed in **C++** as the semester project for the **Programming Fundamentals** course.

The project was designed to demonstrate fundamental programming concepts by providing a simple system for managing campus-related information through a menu-driven interface.
