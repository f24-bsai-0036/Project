# Ink & Code

## Overview

**Ink & Code** is a GUI-based educational programming game developed in **Java** using **Apache NetBeans** as the semester project for the Object Oriented Programming course.

The project applies object-oriented programming principles through an interactive application designed to provide an engaging programming experience.

## Features

- Graphical User Interface (GUI)
- Interactive gameplay
- Object-oriented design
- User interaction

## Technologies Used

- Java
- Apache NetBeans

## Learning Outcomes

- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- GUI Development
- Event-driven Programming
