# Aqua Botanica

## Overview

**Aqua Botanica** is an automated plant irrigation system developed using **Arduino** and electronic components as the semester project for the Digital Logic Design course.

The system monitors soil moisture and automates watering to support efficient plant care.

## Features

- Automatic irrigation
- Soil moisture sensing
- Arduino-based control
- Electronic hardware integration

## Technologies Used

- Arduino
- Sensors
- Electronic Components

## Learning Outcomes

- Digital logic concepts
- Embedded systems
- Sensor integration
- Hardware-software interaction
