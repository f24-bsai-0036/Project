# Project Alimenta

## Overview

**Project Alimenta** is a personalized food recommendation engine developed in **Python** as the semester project for the **Machine Learning** course.

The system recommends food items based on user preferences, demonstrating the application of machine learning techniques in recommendation systems.

## Features

- Personalized food recommendations
- User preference analysis
- Data preprocessing
- Recommendation generation

## Technologies Used

- Python
- PyCharm
- Pandas
- NumPy
- Scikit-learn
