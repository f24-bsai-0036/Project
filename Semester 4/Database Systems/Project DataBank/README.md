# Project DataBank

## Overview

**Project DataBank** is a banking database management system developed as the semester project for the **Database Systems** course.

The project demonstrates database design, SQL implementation, and a simple web-based interface for managing banking records.

## Features

- Customer management
- Account management
- Transaction records
- SQL database operations
- Web-based interface

## Technologies Used

- SQL
- MySQL
- phpMyAdmin
- XAMPP
- HTML
- Visual Studio Code

## Learning Outcomes

Through this project, I gained practical experience with:

- Relational Database Design
- SQL Queries
- Database Normalization
- CRUD Operations
- Database Administration
- Web-Database Integration
