# Database Systems

This directory contains coursework and the semester project completed for the **Database Systems** course.

## Course Overview

This course introduced the principles of database design, relational database management systems, SQL programming, normalization, and database implementation.

## Topics Covered

- Entity-Relationship (ER) Modeling
- Database Design
- SQL
- Normalization
- Joins
- Stored Procedures
- Transactions
